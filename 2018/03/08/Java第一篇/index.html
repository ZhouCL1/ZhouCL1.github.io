代码案例理解

public class HelloWorld { public static void main(String[] args) { System.out.println("HelloWorld"); } }
public:权限修饰符,可以修饰类,方法,变量,常量,关键字字母全部小写;
class:修饰符,声明该类是一个普通Java类(不是接口),关键字字母全部小写;
HelloWorld:这里是类名,类名首字母大写,多个单词每个单词首字母大写,文件名必须和类名一致;
static:静态修饰符,一般用来修饰成员变量或函数。
静态就是指在编译后所分配的内存会一直存在，直到程序退出内存才会释放这个空间。
static不能修饰类,特殊用法用static修饰内部类，普通类是不允许声明为静态的，只有内部类才可以。被static修饰的内部类可以直接作为一个普通 类来使用，而不需实例一个外部类;
static修饰的函数随类的加载而加载,只加载一次;
静态方法只能访问静态成员,且静态方法不能调用this,super,因为静态优先于对象存在;
static修饰的成员变量,能够直接通过类名.变量名来访问到这个变量,没有静态修饰的成员变量需要创建对象(获取实例化对象ClassA a = new ClassA();获取字节码对象三种方式,Class.forName(全限定名);类名.class;对象.getClass())后,通过对象.变量名才能访问;
void:函数没有具体的返回类型;
main:入口函数,是一个特殊的方法名(JVM识别);
String[]:参数类型;
args:参数名;
System.out.println(“HelloWorld”):换行输出打印这个字符串;
